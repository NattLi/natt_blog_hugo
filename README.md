> 个人随笔，防止自己痴呆忘记了年轻时的蠢事。不管怎样，脑子还有输入输出的话，就随便记两笔。

这是我的个人博客代码仓库。记录了从 2006 年至今的一点一滴。

## 🚀 技术栈

- **静态网站生成器**: [Hugo](https://gohugo.io/)
- **主题**: [Hugo Theme Stack](https://github.com/CaiJimmy/hugo-theme-stack) - 一个非常纯粹、卡片式的 Hugo 主题。
- **托管与部署**: [Cloudflare Pages](https://pages.cloudflare.com/) - 极速、免费、稳定的全球分发。
- **代码管理**: [GitHub](https://github.com/)

## 📝 博客内容

这个博客是我思考的自留地，主要涵盖以下领域：

- **生活 (Life)**: 家庭、孩子以及日常琐事。
- **技术 (Tech)**: 各类技术折腾笔记、工具使用与开发心得。
- **艺术设计 (Art & Design)**: 工业设计、交互设计 (UX)、服务设计相关的专业沉淀。
- **观点思考 (Thinking)**: 对世界的观察、思考与复盘。
- **影音娱乐 & 读书随笔**: 那些打动过我的书、电影与音乐。
- **旅行摄影 (Travel)**: 走过的路，看过的风景。
- **创作记录**: 博客本身的迭代与折腾记录。
